| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| NEXT_PUBLIC_ for API keys | Exposes server secret to client bundle | blocking | Only use `process.env.WEATHER_API_KEY` (no NEXT_PUBLIC_ prefix) |
